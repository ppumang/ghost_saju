## 1. 천간 (天干) — 10 Heavenly Stems

| 천간 | 한자 | 오행 | 음양 |
|------|------|------|------|
| 갑   | 甲   | 목   | 양   |
| 을   | 乙   | 목   | 음   |
| 병   | 丙   | 화   | 양   |
| 정   | 丁   | 화   | 음   |
| 무   | 戊   | 토   | 양   |
| 기   | 己   | 토   | 음   |
| 경   | 庚   | 금   | 양   |
| 신   | 辛   | 금   | 음   |
| 임   | 壬   | 수   | 양   |
| 계   | 癸   | 수   | 음   |

### 천간합 (天干合)
- 갑+기 → 토 (갑기합토)
- 을+경 → 금 (을경합금)
- 병+신 → 수 (병신합수)
- 정+임 → 목 (정임합목)
- 무+계 → 화 (무계합화)

**규칙**: 양간과 음간이 5간 간격으로 합. 결과 오행은 고전적 합화 규칙.

## 3. 지지 관계

### 3.1 삼합 (三合)
세 지지가 모여 하나의 오행 기운을 형성.

| 삼합 조합      | 결과 오행 |
|----------------|-----------|
| 인 + 오 + 술   | 화국      |
| 사 + 유 + 축   | 금국      |
| 신 + 자 + 진   | 수국      |
| 해 + 묘 + 미   | 목국      |

**반삼합**: 3개 중 2개만 있을 때. 해당 오행을 지향하는 불완전한 합.

### 3.2 육합 (六合)
두 지지가 합하여 새 오행을 형성.

| 육합 쌍 | 결과 오행 |
|----------|-----------|
| 자 + 축  | 토       |
| 인 + 해  | 목       |
| 묘 + 술  | 화       |
| 진 + 유  | 금       |
| 사 + 신  | 수       |
| 오 + 미  | 화       |

### 3.3 방합 (方合)
같은 방향의 세 지지가 모여 그 방향의 오행을 극대화.

| 방합 조합      | 방향 | 결과 오행 |
|----------------|------|-----------|
| 인 + 묘 + 진   | 동방 | 목국      |
| 사 + 오 + 미   | 남방 | 화국      |
| 신 + 유 + 술   | 서방 | 금국      |
| 해 + 자 + 축   | 북방 | 수국      |

### 3.4 충 (冲)
180도 반대 방향의 두 지지. 충돌/갈등/변동.

| 충 쌍 |
|--------|
| 자 ↔ 오 |
| 축 ↔ 미 |
| 인 ↔ 신 |
| 묘 ↔ 유 |
| 진 ↔ 술 |
| 사 ↔ 해 |

### 3.5 형 (刑) — 4종

**무은지형 (無恩之刑)**: 인↔사, 사↔신, 인↔신
- 은혜를 모르고 배반하는 형벌. 법적 분쟁, 배신.

**지세지형 (恃勢之刑)**: 축↔술, 술↔미, 축↔미
- 세력을 믿고 함부로 하는 형벌. 권력 남용, 독선.

**무례지형 (無禮之刑)**: 자↔묘
- 예의를 무시하는 형벌. 윤리/도덕 문제.

**자형 (自刑)**: 진↔진, 오↔오, 유↔유, 해↔해
- 스스로를 해치는 형벌. 자해/자멸 경향.

### 3.6 파 (破)
약한 수준의 파괴/균열. 충보다 약하지만 관계에 금이 가는 작용.

| 파 쌍 |
|--------|
| 자 ↔ 유 |
| 축 ↔ 진 |
| 인 ↔ 해 |
| 묘 ↔ 오 |
| 사 ↔ 신 |
| 미 ↔ 술 |

### 3.7 원진 (怨嗔)
깊은 원한과 서운함의 관계.

| 원진 쌍 |
|----------|
| 자 ↔ 미 |
| 축 ↔ 오 |
| 인 ↔ 유 |
| 묘 ↔ 신 |
| 진 ↔ 해 |
| 사 ↔ 술 |

## 4. 공망 (空亡)

60갑자는 6개 순(旬)으로 나뉜다. 10천간은 12지지를 모두 채우지 못하므로 매 순마다 2개 지지가 비게 된다. 이것이 공망.

**알고리즘**:
1. 일주의 천간 인덱스(ganIdx)와 지지 인덱스(jiIdx)를 구한다.
2. 순(旬) 시작 지지 인덱스 = (jiIdx - ganIdx + 12) % 12
3. 공망 = 지지[(시작 + 10) % 12], 지지[(시작 + 11) % 12]

**해석**: 공망에 해당하는 기둥(시기)의 일은 허무해지거나 결과가 약하다. 단, 정신/학문/종교에는 오히려 유리.

## 5. 신살 (神殺) — 16종

### 기본 3종 (삼합 기반)
- **도화살**: 일지의 삼합그룹 → 다음 지지 (매력/이성)
- **역마살**: 일지의 삼합그룹 → 충 지지 (이동/변동)
- **화개살**: 일지의 삼합그룹 → 이전 지지 (영성/학문)

### 추가 8종 (v2)
- **반안살**: 일지 + 1 (안장 위 → 이동 위험)
- **육해**: 6쌍 지지 (해가 되는 관계)
- **현침살**: 자/오/묘/유 존재 (날카로움)
- **귀문관살**: 일지 → 특정 지지 (귀신의 문)
- **지살**: 년지 삼합 → 특정 지지 (재난)

### 추가 8종 (v3)
- **양인살**: 양간의 건록 다음 지지 (극강한 기운)
- **괴강살**: 무진/무술/경진/경술 일주 (강렬한 카리스마)
- **백호살**: 일지 → 충 지지 (혈광)
- **겁살**: 년지 삼합 역방향 (겁탈)
- **천라지망**: 진+술 또는 사+해 동시 존재 (하늘의 그물)
- **망신살**: 년지 기준 (명예 실추)
- **장성살**: 년지 기준 (장군의 별)
- **공망살**: 공망 결과 연동 (공허)

## 7. 격국 (格局) — Pattern Classification

### 판정 알고리즘
1. 월지 정기(본기)의 천간을 구한다.
2. 일간 기준으로 그 천간의 십신을 판정한다.
3. 해당 십신이 격국이 된다.

### 월지 정기 테이블
| 월지 | 정기 천간 |
|------|-----------|
| 자   | 계       |
| 축   | 기       |
| 인   | 갑       |
| 묘   | 을       |
| 진   | 무       |
| 사   | 병       |
| 오   | 정       |
| 미   | 기       |
| 신   | 경       |
| 유   | 신       |
| 술   | 무       |
| 해   | 임       |

### 내격 8격
식신격, 상관격, 편재격, 정재격, 편관격, 정관격, 편인격, 정인격

### 특수격
- **건록격**: 월지 정기가 비견일 때. 자수성가의 격.
- **양인격**: 월지 정기가 겁재일 때. 극단적 에너지.

### 종격 (從格)
사주 전체의 오행/십신이 극단적으로 치우쳐 일간이 세력에 따르는 격.

| 종격 | 조건 |
|------|------|
| 종강격 | 비겁+인성 80%+ & 재성=0 & 관성=0 |
| 종아격 | 식상 60%+ & 인성=0 |
| 종재격 | 재성 60%+ & 비겁=0 |
| 종관격 | 관성 60%+ & 식상=0 |
| 종세격 | 식상+재성+관성 80%+ & 비겁=0 |

## 8. 강약 판정

### 득령 (得令)
일간의 오행이 월지 계절에서 왕성한지 판정.
- 월지의 왕성 오행 = 일간 오행이면 득령
- 월지의 왕성 오행이 일간을 생하면 득령

### 강약 판정
비겁+인성(help)의 비율로 판정:
- 70+ → 태강
- 55~69 → 강
- 45~54 → 중화
- 30~44 → 약
- 29 이하 → 태약

득령 시 +10 보너스.

## 9. 용신 (用神)

억부법(抑扶法) 기반:

| 강약 | 용신 | 희신 | 기신 |
|------|------|------|------|
| 강/태강 | 설기(식상) = 일간이 생하는 오행 | 극(관성) = 일간을 극하는 오행 | 인성 = 일간을 생하는 오행 |
| 약/태약 | 생(인성) = 일간을 생하는 오행 | 비겁 = 같은 오행 | 관성 = 일간을 극하는 오행 |

### 용신 속성 매핑
| 오행 | 색상 | 방향 | 숫자 | 계절 | 맛 |
|------|------|------|------|------|-----|
| 목 | 초록/청색 | 동쪽 | 3, 8 | 봄 | 신맛 |
| 화 | 빨강/주황 | 남쪽 | 2, 7 | 여름 | 쓴맛 |
| 토 | 노랑/갈색 | 중앙 | 5, 10 | 환절기 | 단맛 |
| 금 | 흰색/은색 | 서쪽 | 4, 9 | 가을 | 매운맛 |
| 수 | 검정/남색 | 북쪽 | 1, 6 | 겨울 | 짠맛 |

| 내가 극하는 | 편재 | 정재 |

## 11. 십이운성 (十二運星)

일간의 기운이 12단계 생사 사이클을 거치는 과정.

장생 → 목욕 → 관대 → 건록 → 제왕 → 쇠 → 병 → 사 → 묘 → 절 → 태 → 양
